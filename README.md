# 🌱 Productivity Hub

A modern productivity web app that combines **task management**, a **focus timer**, and a **visual growth system** to help you stay consistent and motivated.

---

## 🚀 Live Demo

🌐 **Try it here:**
👉 https://productivityhhu.netlify.app/

---

## ✨ Features

This app combines **productivity + gamification + clean UX** into one experience.

### ✅ Task Manager

* Add, delete, and complete tasks
* Priority-based system (High / Medium / Low)
* Real-time updates

### 🌿 Focus Garden

* Pomodoro-style focus timer
* Plant grows as you stay focused
* Persistent progress tracking

### 🧠 AI Productivity Coach

* Smart task suggestions
* Helps prioritize your workflow
* Works instantly (no external API required)

---

## ⏱️ Timer Controls

* Start focus session
* Pause / Resume
* Reset timer
* Accurate second-based tracking

---

## 📊 Dashboard

* Task completion percentage
* Pending vs completed tasks
* Visual analytics using charts

---

## 💾 Data Persistence

* Uses **localStorage**
* Automatically saves tasks, sessions, and progress
